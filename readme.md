# Data Cleaning & Sales Analysis Project

## Overview
This project focuses on cleaning and analyzing a retail sales dataset to improve data quality and extract useful business insights.

## Objectives
- Clean and standardize messy data
- Transform raw data into structured format
- Analyze sales and profit trends

## Data Cleaning Steps
- Standardized column names
- Converted date columns to datetime format
- Removed duplicate records
- Dropped unnecessary columns
- Created new features (order_year, order_month)

## Analysis Performed
- Sales by category
- Profit by region
- Monthly sales trends
- Profit by sub-category

## Key Insights
- Technology category generated the highest sales
- West region produced the highest profit
- Some sub-categories show low or negative profit

## Tools Used
- Python
- Pandas

## Output
- Cleaned dataset (`cleaned_sales.csv`)
- Insights for business decision-making
